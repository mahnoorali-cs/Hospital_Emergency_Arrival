# Hospital_Emergency_Arrival
Project Title:
      Hospital Emergency Arrival

Project Overview:
Predicting the class of patients either critical or non critical using logistics regression and Distributions to predict the probabilities of critical class and predicting the number of patients each hour. 

Dataset:
PatientId,	PatientName,	PatientAge,	BloodPressure,	HeartRate,	Temp,	critical,	ArrivalTime,	Department

Libraries and tools:
Python langauage
Numpy, Pandas, Matplotlib, Seaborn, Scikit-Learn, scipy, stats, datetime
Google Colab

Models:
Logistic Regression
Binomial, Poisson Distribution 

Evaluation:
Accuracy Score, Confusion Matrix, Visualization Graphs _ histplot, barplot
